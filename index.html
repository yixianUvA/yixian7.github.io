Research

I am broadly interested in machine learning security, privacy, and application. My current interests include the following domains:

  • Privacy Preserving Machine Learning: the analysis and application of backdoors in machine learning models (ACSAC'19)
  • Machine Learning for Cyber Security: machine learning based steganography preserving communication(HPCC'19)
  • Machine Learning Security: attacks and defenses of machine learning
